# Fonts are omitted on purpose

This deck is designed with Studytube's brand fonts — **Aeonik** (display) and
**Euclid Circular B** (body). Those are commercially licensed and are **not**
redistributed in this repo.

The deck falls back to a clean system sans-serif stack automatically, so it
renders fine as-is. If you have the brand fonts installed locally (or open the
deck from the Studytube design-system export), you'll get the exact brand type.

**Do not commit the font binaries (`.ttf` / `.otf`) to this repo.**
